调用方法：
var llist = require("./linkedlist");
var linklist = new llist.linklist();

linklist.init();
linklist.insert({edit_id:"1234",dataURI:"5678"});

使用方法：
--------------------------------------------
node结构：
var Node = function (pData) {
  this.next = null;
  this.prev = null;
  this.data = pData;
}

参数
 pData：Array

在新建node时可将参数传入。
例：var newNode = new Node(data);
--------------------------------------------
init (pData)

参数
  pData：Array

初始化linklist的head和tail。
--------------------------------------------
insert (pData) 

参数
  pData： Array

在末尾插入新的Node，tail将设为此Node，且Node.data
为pData。
--------------------------------------------
getData (edit_id) 

参数
  edit_id：string

返回edit_id下的Node中的data。
--------------------------------------------
getNode  (edit_id) 

参数
  edit_id：string

返回edit_id下的Node。
--------------------------------------------
removeAt (edit_id) 

参数
  edit_id：string

删除edit_id下的Node。
--------------------------------------------
getTail () 

返回尾节点Node。
--------------------------------------------
getHead ()

返回头节点Node。
--------------------------------------------
getSize ()

返回linklist中Node的个数。（包含head和tail）
--------------------------------------------
createFromArray (head,array)

参数
  head：Node
  array：Array

从一个数组创建linklist,需要传入一个Node作为初始化后
的head。
--------------------------------------------
print = function () 

顺序打印所有Node的data。
--------------------------------------------
printFromBack = function () 

逆序打印所有Node的data。
